Portal Session Capture

This is the login bridge for live purchase pricing. Capture each logged-in portal once, then the purchase console can use those sessions for quote adapters.

Install in Chrome

  1. Open Chrome extensions Go to chrome://extensions and enable Developer mode.
  2. Load unpacked Select /tmp/hn-hotels-purchase-console/ext/purchase-portal-capture.
  3. Open each portal tab Use your already logged-in Chrome tabs for Hyperpure, Zepto, Flipkart Minutes, Instamart, Blinkit, Amazon Now, BigBasket, and JioMart.
  4. Capture current tab Enter the purchase-console PIN and click capture. The popup sends the session to the secure vault and shows only counts/status.
  5. After extension updates Click the reload icon for the extension in chrome://extensions, then recapture Hyperpure so live SKU pricing can read its browser-visible session cookies.
  6. Run browser live quotes After clicking Get quotes in the purchase console, keep the logged-in portal tab open and click Run live quotes from tab in the extension.

What Phase 5 does

It solves the login/token layer and lets the browser execute live quote jobs when a portal rejects server-side calls. Hyperpure is the first live runner; the remaining portals follow the same source contract.
