How to read this report:
Every day, we buy chicken in 8 cuts from MN Broilers.
Each dish on the menu uses a specific cut. The recipe says how many grams each dish needs.
Should Use = recipe grams × how many we sold. Actually Used = what we bought (divided proportionally across dishes).
If "Actually Used" is more than "Should Use" — the extra is waste, marinade weight, or bigger portions.
Sales from 3 channels: POS (dine-in/takeaway) + Swiggy [S] + Zomato [Z] aggregator orders.
