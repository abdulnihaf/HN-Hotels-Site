Hybrid scoring formula + sector cap algorithm 
hybrid_score = rr_ratio × green_day_rate × log(liquidity_score+1) × recency_factor

For top 30 candidates by hybrid_score:
  for each pick (max 3):
    select highest hybrid_score where sector_bucket NOT IN already-picked
    if conviction = HIGH: weight = 35%
    elif MED:            weight = 30%
    else LOW:            weight = 25%
  normalize weights to ≤ 95% (5% slippage buffer)

LLM fallback: Opus → Sonnet → Haiku → engine_only (mechanical R:R rank)

Exact P&L formula at exit

gross_pnl = (exit_paise − entry_paise) × qty

fees = ₹40 brokerage
       + sell_turnover × 0.025% (STT)
       + (buy + sell turnover) × 0.00322% (exchange)
       + buy_turnover × 0.003% (stamp)
       + (brokerage + exchange) × 18% (GST)
       → ~0.04-0.06% round-trip (Zerodha intraday MIS realistic)

net_pnl_paise = gross_pnl − fees
win_loss = net > 0 ? 'win' : net < 0 ? 'loss' : 'breakeven'
